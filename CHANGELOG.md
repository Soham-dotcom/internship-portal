### Technical Stack
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Frontend:** React 19, TailwindCSS 4, React Router 7
- **Charts:** Recharts 3
- **Excel:** XLSX
- **HTTP:** Axios
- **Upload:** Multer
